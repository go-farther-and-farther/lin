# lin

#### 介绍
一个Yunzai-BotV3.0+的插件,提供一些小玩具


### 使用说明

- 推荐使用git进行安装，以方便后续升级。在BOT根目录夹打开终端，运行

`git clone https://gitee.com/go-farther-and-farther/lin ./plugins/lin/`

使用  #lin更新  来更新插件

### 现有功能

#lin帮助 获取帮助

|#御前决斗@......|与群友一起决斗|  
    
|#锻炼|增加在决斗中胜出的可能性|

|#我的境界|查看自己的战斗力和境界|

|#设置半步管理员@......|让某位群友进入无敌状态|

#点赞部分

自动点赞

#消息回复部分

自动回复消息

#点歌部分

点歌+...
